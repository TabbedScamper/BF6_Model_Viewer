How these models are sourced

Every model here is the actual mesh from Battlefield 6 / Portal, extracted straight from the game's own asset files — nothing remodelled.

Each prop's high-poly MeshSet is converted to geometry with correct UVs, and its textures are pulled from the same asset folder and assigned by the role declared in each texture's own header (World_BaseColor, World_Normal). Colour and normal maps are decoded from BC7 and applied exactly as the game binds them, giving a pixel-perfect surface for direct-textured props.

Models are shown at web-optimised texture resolution for fast browsing. Meshes and textures are from Battlefield 6 / Portal (© EA & DICE), extracted for reference and modding use.

Install the High-Poly Library

This library feeds the High-Poly Godot plugin for the BF6 Portal SDK — it swaps the editor's low-poly placeholders for the real game models while you build. You only download it once.

Download Library or use the plugin's built-in Download Full Library button (recommended — it installs itself)

Recommended — let the plugin do it

  1. Install the High-Poly plugin in your Portal SDK Godot project and enable it (Project → Project Settings → Plugins).
  2. Open the High-Poly dock and click Download Full Library. It downloads and installs everything automatically — no manual unzipping.
  3. Let the first import finish (Godot imports thousands of models once; this is slow only the first time).
  4. Pick Medium or High-Poly mode and build. Done.

Manual install (from this download)

  1. Download the .zip above.
  2. Extract it into your Godot project folder so the models land in res://highpoly/ (the zip already contains the highpoly/ folder — extract it at the project root).
  3. Back in Godot, let the import finish, then use the plugin as above.
You only download the full pack once. After that, whenever models are corrected or added, just click Update Models in the plugin dock — it compares your local files against the registry and downloads only what changed (usually a few small files, not the whole pack).
